Codelist Tools

March 20, 2024

A set of packages for better codelist management in medical research, including support for SNOMED, OPCS and ICD-10 classification systems. Allows for validation of codelists and mapping between different classification systems.

The core functionality is written in Rust for fast and memory-efficient operations. Python and R bindings allow for easy integration into existing workflows.
